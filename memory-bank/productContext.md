## Project Goal

为孩子开发一个纯HTML的积分展示系统，用于记录和展示孩子获得的积分，支持积分的增加和减少操作。

## Key Features

* 纯HTML/CSS/JavaScript实现，无需服务器
* 本地存储积分数据（localStorage）
* 简洁友好的用户界面，适合孩子使用
* 积分增加/减少功能
* 积分历史记录显示
* 响应式设计，支持手机和电脑

## Overall Architecture

* 前端：单一HTML文件，包含CSS和JavaScript
* 数据存储：浏览器本地存储（localStorage）
* 用户界面：响应式设计，适配多种设备
* 数据持久化：自动保存积分变更到本地存储
